## ✨ 主要功能

- 📅 赛程查看（按联赛/日期筛选）
- ⭐ 主队收藏，比赛前自动闹钟提醒
- 📊 积分榜实时更新
- 🏆 赛果/比分查询
- 🏟 世界杯分组对阵图
- 🔔 赛前通知（可选 15min / 30min / 1h / 24h）

## 🔧 技术栈

- **前端**: Kotlin + WebView (`file:///android_asset/`)
- **数据源**: football-data.org / 聚合数据 API
- **服务端**: Node.js (阿里云 ECS, PM2)
- **CDN**: Gitee Release 分发
- **离线**: 静态数据兜底，保证无网络可用

## 📥 安装方法
